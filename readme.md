## Run Locally

Clone the project

```bash
  git clone https://github.com/abhaypsamrat/React-Dashboard.git
```

Go to the project directory

```bash
  cd React-Dashboard
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
